FAQ

Pertanyaan yang sering ditanyain sebelum mulai project.

Untuk logo biasanya 2–4 hari kerja, tergantung kompleksitas & jumlah revisi. Untuk paket konten bisa 3–7 hari.

Tergantung paket yang dipilih. Paket premium biasanya revisi lebih fleksibel, selama masih dalam satu konsep.

Minimal kamu dapat file PNG, JPG resolusi tinggi, plus PDF siap cetak. Untuk logo bisa juga disertakan file vector (AI / EPS / SVG) jika dibutuhkan.
